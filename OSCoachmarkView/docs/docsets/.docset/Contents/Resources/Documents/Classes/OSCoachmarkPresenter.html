OSCoachmarkPresenter

public class OSCoachmarkPresenter

OSCoachmarkPresenter* handles everything related to showing, hiding and managing the states of a coachmark.

The view property is of utmost importance. This class assumes you have supplied the necessary UIView subclass which this presenter applies it’s logic on.

Remark

This class is to be used to display any UIView subclass with behaviour of a coachmark

Requires

A UIView subclass

Warning

A wonderful crash will be the result of a nil value for the public view property

Version

0.1.0

Author

Aamir Anwar
